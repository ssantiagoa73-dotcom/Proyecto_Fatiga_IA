# Sistema de Detección de Fatiga con IA

Proyecto de visión artificial y machine learning para detectar fatiga, distracción y concentración en tiempo real utilizando OpenCV, MediaPipe y Random Forest.

## Tecnologías utilizadas

- Python
- OpenCV
- MediaPipe
- Scikit-learn
- Pandas
- Matplotlib

## Funciones

- Detección facial
- Detección de ojos
- Detección de iris
- Detección de cabeceos
- Detección de microsueños
- Detección de bostezos
- Predicción IA en tiempo real

## Resultados

Accuracy del modelo: 96%

## Ejecución

Captura de datos:

```bash
python captura_datos.py
```

Detección en tiempo real:

```bash
python modelos/deteccion_tiempo_real.py
```
